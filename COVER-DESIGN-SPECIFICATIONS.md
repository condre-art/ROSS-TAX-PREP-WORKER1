## DESIGN ASSET SPECIFICATIONS

### Logo

**ROSS Tax Academy Shield**:
- **Shape**: Rounded square shield
- **Primary Color**: Gold (#B8860B)
- **Secondary Color**: Teal (#1a4d7a)
- **Center Text**: "ROSS" (white, bold)
- **Sizing**: 
  - Large: 2" × 2" (cover front)
  - Medium: 1.5" × 1.5" (back cover)
  - Small: 0.5" × 0.5" (footer)

---

### Typography

**Headers** (Trebuchet MS Bold):
- H1 (Title): 36pt, Dark Gray or Gold
- H2 (Chapter): 24pt, Teal
- H3 (Section): 16pt, Dark Gray
- H4 (Subsection): 12pt, Dark Gray

**Body Text** (Calibri Regular):
- Standard: 11pt, Dark Gray
- Emphasis: 11pt Italic, Teal
- Code: 9pt, Courier New, #333333

---

### Icons & Design Elements

**Checkmarks** (✓):
- Color: Gold (#B8860B)
- Use: Learning outcomes, feature lists
- Size: 12pt (inline), 16pt (standalone)

**Borders**:
- Color: Teal (#1a4d7a) primary, Gold secondary
- Width: 2pt (thick lines), 1pt (dividers)
- Style: Solid lines, rounded corners

**Page Breaks**:
- Color: Teal with gold accents
- Design: Horizontal line with centered text/icon
- Spacing: 0.5" top and bottom

## DESIGN DELIVERY CHECKLIST

### For Each Cover

**Front Cover**:
- ☐ Correct trim size with bleed area (0.125" all sides)
- ☐ 300 DPI resolution (for print)
- ☐ CMYK color space (for offset printing) or RGB (for digital)
- ☐ All text converted to curves or embedded fonts
- ☐ ROSS logo properly sized and positioned
- ☐ "Train. Certify. Excel." tagline included
- ☐ Title, subtitle, course code, ISBN visible
- ☐ Author name and PTIN included
- ☐ Edition year (2025-2026) clearly stated
- ☐ File format: PDF or PSD (layered)

---

**Back Cover**:
- ☐ Course description (150-200 words)
- ☐ Learning objectives bullet list
- ☐ Perfect for... section (target audience)
- ☐ ISBN barcode (bottom right corner, 1" × 0.5")
- ☐ Publisher info and website
- ☐ Contact information
- ☐ Proper bleed area
- ☐ 300 DPI resolution
- ☐ CMYK or RGB color space
- ☐ Barcode scannable (no interference)

---

**Spine**:
- ☐ Text readable when book standing upright
- ☐ Publisher logo at bottom
- ☐ Title readable
- ☐ Proper width for page count
- ☐ No text cutoff

---

### Assembly File (Front + Spine + Back)

- ☐ Correct total width (front + spine + back + 0.25" bleed)
- ☐ Height matches trim size exactly
- ☐ Spine width calculated for page count
- ☐ Elements properly aligned and registered
- ☐ All bleed areas present
- ☐ File format: PDF or PSD
- ☐ Resolution: 300 DPI
